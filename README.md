# bunrun

## Quick Start

### 1. Install Project

```console
$ git clone https://github.com/bunrun-app/bunrun
```

### 2. Open in Xcode

```console
$ cd bunrun
$ open bunrun.xcodeproj
```

### 3. Build & Run

Select a target device in Xcode, and click the "Run" button or <kbd>Cmd</kbd> + <kbd>R</kbd> to build and run the app.
